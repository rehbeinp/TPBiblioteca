# TPBiblioteca
Trabalho prático de Linguagem de Programação para WEB - SI/UFSM-FW

## Requisitos Mínimos do Sistema ENUNCIADO DE TRABALHO:
### Menus de Navegação 
O sistema deve possuir menus claros e organizados para facilitar a navegação do usuário. 
* Menu principal com acesso às principais funcionalidades  
* Submenus, quando necessário  
* Diferenciação de menus conforme o perfil (ex: administrador x usuário comum) 

### Formulários 
* O sistema deve conter formulários para entrada de dados.  Cadastro (ex: usuários, produtos, alunos, etc.)  
* Edição e exclusão de dados  
Validação dos campos (obrigatórios, formatos, etc.)  
Feedback ao usuário (mensagens de erro ou sucesso) 

### Login de Usuário com Validação Controle de acesso ao sistema.  
* Tela de login com usuário e senha  
* Validação dos dados no backend (PHP)  
* Senhas armazenadas de forma segura (hash)  
* Mensagens de erro para login inválido  
* Controle de acesso por nível (ex: administrador, usuário) 

### Tela de Gerenciamento (Administrador) 
* Área restrita para controle do sistema. 
* Cadastro, edição e exclusão de registros  
* Gerenciamento de usuários  
* Visualização geral dos dados  
* Controle de permissões  
* Interface organizada para facilitar a administração 

### Tela de Relatórios
* Apresentação de informações do sistema.  
* Listagem de dados cadastrados  
* Filtros (por data, nome, categoria, etc.)  
* Possibilidade de ordenação  
* Exibição clara (tabelas)  
* (Opcional) Exportação para PDF ou Excel 

### Banco de Dados Armazenamento das informações do sistema.  
* Utilização de banco relacional (ex: MySQL)  
* Tabelas estruturadas corretamente  
* Relacionamentos entre tabelas (chaves estrangeiras)  
* Uso de comandos SQL (SELECT, INSERT, UPDATE, DELETE) 

### Uso de Sessões ou Cookies 
* Gerenciamento da autenticação do usuário.  
* Uso de $_SESSION para manter o usuário logado  
* Proteção de páginas restritas  
* Logout do sistema (encerramento da sessão)  
* (Opcional) Cookies para lembrar usuário 
* Menu pro usuário e o adm
